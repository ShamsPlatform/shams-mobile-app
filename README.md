# ☀️ منصة شمس | Shams Platform

![Status](https://img.shields.io/badge/Status-In--Development-blue.svg)

**شمس** هي منصة اجتماعية وتقنية متكاملة صُممت خصيصاً لخدمة سوق الطاقة الشمسية في اليمن. تهدف المنصة إلى سد الفجوة بين الفنيين والورش والعملاء في بيئة آمنة وموثقة.

---

## 🚀 الميزات الرئيسية (Core Features)

* **🛡️ توثيق الهوية (Identity Verification):** نظام صارم لتوثيق هويات الفنيين والورش لضمان الموثوقية.
* **🛠️ إدارة الورش (Workshop Management):** واجهة مخصصة للفنيين لعرض خدماتهم وتخصصاتهم (تركيب، صيانة، برمجة).
* **📱 السجل الاجتماعي (Social Feed):** مساحة لمشاركة الإنجازات الميدانية، النصائح التقنية، والتوثيق البصري (قبل وبعد).
* **🔔 نظام الإشعارات الذكي:** متابعة لحظية لطلبات الصيانة وتفاعلات المجتمع.
* **💬 الدردشة والطلبات:** ربط مباشر بين العملاء والورش لتسهيل تقديم الخدمات.
